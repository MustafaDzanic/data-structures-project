Student API - README file
Date: 25.06.2017

Student API is part of an university project on the Faculty of Engineering in Tuzla. It includes the database API and the API interface source files.

/API contains the full CodeBlocks project
/API/db contains the database files
/API/dto contains the Data Transfer Object header file
You can also find images of the API's and database's UML representation.
